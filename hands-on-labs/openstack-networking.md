# OpenStack Networking (Neutron)

*Hands-On Lab: Configuring advanced networking with OpenStack Neutron*

---

## Lab Overview

Deep dive into OpenStack Neutron networking, covering virtual networks, routers, security groups, and advanced networking features for multi-tenant cloud infrastructure.

---

## Topics Covered

- Neutron architecture
- Provider networks
- Tenant networks and isolation
- Virtual routers
- Floating IP addresses
- Security groups and firewall rules
- Load balancing
- VPN as a Service
- Network troubleshooting

---

## Neutron Components

### Core Services
- **Neutron Server:** API and orchestration
- **L3 Agent:** Virtual routing
- **DHCP Agent:** IP address management
- **Metadata Agent:** VM metadata service
- **OVS Agent:** Open vSwitch integration

### Network Types
- **Provider Networks:** Direct external connectivity
- **Tenant Networks:** Isolated private networks
- **External Networks:** Public IP access
