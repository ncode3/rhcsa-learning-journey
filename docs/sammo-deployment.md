# SAMMO Fight IQ Deployment Guide

*Production AI deployment on bare metal infrastructure*

## Overview

SAMMO Fight IQ is an AI-powered boxing coach that analyzes sparring footage using computer vision (MediaPipe) and provides personalized coaching feedback.

**Status:** Live on Dave's GDC infrastructure

---

## Architecture

- **Backend:** Python/FastAPI
- **Frontend:** React
- **AI/ML:** MediaPipe pose detection, TensorFlow
- **Infrastructure:** OpenShift on KVM/bare metal
- **Storage:** NFS persistent volumes (100GB)
